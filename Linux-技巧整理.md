Linux 技巧整理
=============
---
* `ldd --version`   //查看glibc版本
* `python -m SimpleHTTPServer` //在当前目录开启一个建议的http服务 可用于共享文件
* /etc/fstab       uid=1000,gid=1000,default		//guazaiquanxian
* `id klbj`   //查看用户信息
* `root # gpasswd -a <user> vboxusers`		//把用户添加到组
* `root # netstat -tupln`	// show service port
* `ipcs` //查看共享内存
